1. What is Search it?

Search it is a search web browser like Firefox, google chrome and etc. Search it 1st version are published in the github by Z4RN-CMD(its me :D)

Its also avalaible in the Web variant.

2. How to use Search it?

  1. Type a thing you want to know. example: "How to cook an egg"
  2. After that press enter
  3. You will see the google search of things you want to know!
